# MECHAT 
> Chat App using MERN stack and Socket.io
 
 <hr/>
 
 ## Features
- Authentication
- Real-time chat
- Real-time Notification
- Typing indicator
- Guest login
- Group chat
- One-on-one chat
- Responsive
